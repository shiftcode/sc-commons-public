# airtable-mapper

> 🎯 Target runtime: es2023 ([Browser](https://caniuse.com/?search=es2023) & ([Node >= 20](https://node.green/#ES2023))

Module to create the airtable models and their mapping

Is a zero dependency package, so we can use it in the client\
(if combining with airtable-syncer package, we're no longer able to use actual models in client code)
